This course was recorded using Next.js version 14.

The Next.js team launches new versions very frequently, and while I will try to keep this part up-to-date with the most important features, please understand that I won't re-record these videos each time a new Next.js version drops.

Keeping up with framework updates is actually one of the most fundamental skills of any web developer, so keeping this project up-to-date with Next.js versions is a very good challenge for you

The Next.js documentation is your best friend for this (yes, you need to know how to read and follow documentation as a professional developer ).

To update your project to the latest Next.js version, just run this command in your project folder:

npm install next@latest react@latest react-dom@latest eslint-config-next@latest

You can also follow the Next.js blog where new versions are announced.
